  <action>
  Create src/asebytes/h5md/_store.py with class H5MDStore that implements ColumnarStore.

  **Constructor:**
  ```
  def __init__(self, *, path=None, file_handle=None, file_factory=None,
               group="default", readonly=False, compression="gzip",
               compression_opts=None, chunk_frames=64, rdcc_nbytes=64*1024*1024):
  ```
  - Exactly one of {path, file_handle, file_factory} must be provided
  - If path: open h5py.File, _owns_file=True
  - If file_handle: use directly, _owns_file=False
  - If file_factory: store factory, create context manager wrapper, _owns_file=False
  - Store the particles group name (self._particles_group = group)

  **Core path translation method:**
  `_column_to_h5(self, key: str) -> tuple[str, str | None]`
  Maps flat column names to (h5_path, origin) tuples:
  - "cell" -> ("/particles/{grp}/box/edges", "atoms")
  - "pbc" -> ("/particles/{grp}/box/pbc", "atoms")
  - "arrays.positions" -> ("/particles/{grp}/position", "atoms")
  - "arrays.numbers" -> ("/particles/{grp}/species", "atoms")
  - "arrays.{other}" -> ("/particles/{grp}/{h5md_name}", "arrays")
  - "calc.forces" -> ("/particles/{grp}/force", "calc")
  - "calc.energy" -> ("/observables/{grp}/potential_energy", "calc")
  - "calc.{name}" -> particles if in KNOWN_PARTICLE_ELEMENTS else observables, "calc"
  - "info.{name}" -> ("/observables/{grp}/{name}", "info")
  Use ASE_TO_H5MD for name translation. Return (None, None) for unrecognized keys.

  **Reverse path translation:**
  `_h5_to_column(self, h5_path: str) -> str | None`
  Walks H5MD paths back to flat column names (needed for list_arrays and _discover).

  **H5MD time-dependent element structure:**
  Every "column" in H5MD is actually a group with step/time/value sub-datasets.
  - `create_array(name, data, *, dtype=None, fill_value=None)`: Translate name to h5_path via _column_to_h5. Create group at h5_path. Create `value` dataset inside it (with compression, maxshape all-None, chunks). Create `step` as scalar int dataset (value=1 for linear). Create `time` as scalar float dataset (value=1.0 for linear). Write ORIGIN_ATTR on the group. Write unit attribute on value dataset if applicable.
  - `get_array(name)`: Translate to h5_path, return h5_path/value[()]
  - `get_slice(name, sel)`: Translate to h5_path, return h5_path/value[sel]
  - `append_array(name, data)`: Translate to h5_path, resize h5_path/value and write (same as HDF5Store.append_array but on /value sub-dataset)
  - `write_slice(name, sel, data)`: Write to h5_path/value[sel]
  - `has_array(name)`: Translate to h5_path, check h5_path/value exists
  - `get_shape(name)`: Return shape of h5_path/value
  - `get_dtype(name)`: Return dtype of h5_path/value

  **list_arrays():**
  Walk particles/{grp}/ and observables/{grp}/ recursively. For each H5MD element (group with 'value' dataset), translate back to flat column name via _h5_to_column. Return sorted list.

  **Attrs (get_attrs/set_attrs):**
  Operate on the `asebytes/{grp}` group inside the file (for internal metadata like _n_atoms, _max_atoms, _n_frames, etc.). Create this group if needed on set_attrs.

  **list_groups(path):**
  Static method. Open HDF5 file read-only, look for `particles/` children. Return group names found under particles/.

  **Box handling:**
  The box group is special: /particles/{grp}/box/ has attrs (dimension=3, boundary=[...]) and edges/pbc as sub-elements.
  - "cell" maps to box/edges (time-dependent with step/time/value)
  - "pbc" maps to box/pbc (znh5md extension, time-dependent)
  - On create_array("cell", ...), also set box attrs: dimension=3, boundary=["periodic","periodic","periodic"] (default, can be overridden)

  **Units mapping:**
  Store a dict mapping ASE names to units:
  positions->Angstrom, forces->eV/Angstrom, stress->eV/Angstrom^3, velocities->Angstrom/fs, cell->Angstrom, energy->eV
  Write `ds.attrs["unit"] = unit_str` on the value dataset when creating arrays.

  **close():** Close h5py.File only if _owns_file is True.

  NOTE: Keep the _ds_cache pattern from HDF5Store for performance (cache h5py.Dataset references by column name).
